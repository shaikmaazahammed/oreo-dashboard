# 🍪 The Great Oreo Takeover — Campaign Dashboard

A Streamlit dashboard for the Oreo Hungary brand awareness campaign content calendar.

## Features
- **Content Calendar** — Browse all 43 posts with platform, format, phase, and KPI filters
- **KPI Tracker** — Monitor weekly performance against campaign targets
- **Budget Breakdown** — Visual allocation of the 20M HUF campaign budget

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Community Cloud

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click **New app** → connect your GitHub repo
4. Set **Main file path** to `app.py`
5. Click **Deploy**
